OkFriend

OkCupid but for friends! A place to find, meet, and set up events with friends by area, interest, gender, and age

  • Implemented dynamic routing to successfully generate URLs for over 500 users
  • Used RandomUser.me as an API to populate internal database with information from random users
  • Utilized the zipcodes NPM package to filter users by a certain radius from the current user
  • Integrated Ruby’s RestClient gem to access data from external APIs
  • Utilized JSON Web Tokens and localStorage to store encrypted user information client-side
  • Created a chat feature for users to communicate with one another
  • Users can also connect through calendar invites

Built with:

  • React JS
  • Ruby on Rails

We ♥️ Dogs

Recreated an Instagram-style web application that allowed you to rate and comment on various dog pictures

  • Used the Twitter API to access pictures of dogs from various Twitter accounts
  • Combined various technologies (Twurl, Ruby) to populate database with pictures of dogs from the Twitter API
  • Implemented Fetch in JavaScript to grab pictures of dogs from internal API
  • Utilized custom CSS and CSS Grid to recreate Instagram’s look and feel
  • Used responsive web design that adjusts the user’s experience based on their browser size

Built with:

  • HTML/CSS
