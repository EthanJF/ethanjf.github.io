OkFriend

About:

OkFriend is a social media website that acts like a dating app but is designed to look for friends. It includes a search feature for both users in your area and users who have similar interests. You can create calendar invites with your friends, and there's also a chat feature. It utilizes JWT authorization and is made responsive with CSS Grid.

Built with:

  • React JS
  • Ruby on Rails

We ♥️ Dogs

About:

We ♥️ Dogs is an instagram clone focused just on dogs. I used the Twurl tool to scrape image URLs from various Twitter accounts featuring dogs. You can click on each dog where you can like it, rate it, and leave a comment. You can sort all dogs by popularity, highest rating, and most comments. The site is fully responsive, utilizing CSS Grid.

Built with:

  • HTML/CSS
  • Vanilla JavaScript
  • Ruby on Rails
  • Twurl
